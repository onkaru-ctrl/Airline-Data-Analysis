# ✈️ Airline Data Analysis (Power BI)

## 📊 Project Overview
This project analyzes airline data using Power BI to generate meaningful insights through interactive dashboards.

## 🎯 Objective
To understand airline performance, passenger trends, and revenue patterns using data visualization.

## 🔧 Tools & Technologies
- Power BI
- Excel

## 📈 Key Insights
- Identified revenue trends across different routes
- Analyzed flight delays and performance
- Studied passenger booking patterns
